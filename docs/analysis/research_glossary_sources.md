# 공인 기관 용어집 레퍼런스 조사 결과

2026-07-18 딥 리서치(웹 검색 5축 + 소스 검증 + 3표 교차 검증) 결과 요약이다.
목적은 양자컴퓨팅·암호화·AI·클라우드 등 최신 분야 용어를 glossary.tsv에 반영할
신뢰할 수 있는 출처 선별이다.

## 추천 순위

1. **TTA 정보통신용어사전** (terms.tta.or.kr, 한국정보통신기술협회)
   - 한·영(·한자) 병기. 보안 1,605건, 양자 490건(양자내성암호 PQC 13건), 클라우드
     413건, AI 388건. 신규·갱신 용어 지속 반영.
   - 개별 용어 조회 1순위 참조처. 단 전체 다운로드가 없고 저작권이 All Rights
     Reserved라 **대량 수집·재배포는 금지**. 대량 이용은 terms@tta.or.kr 문의 필요.
   - 검색은 POST(searchCate=bigram)만 동작(GET은 0건 반환).
2. **국립국어원 다듬은 말** (korean.go.kr, 18,323건, 공공누리)
   - 원어→다듬은 말 대응(RAG→검색 기반 생성 등). 라이선스 제약 최소.
   - 순화어 성격이라 업계 통용 표기와 다를 수 있음. 반영 시 note에 '다듬은 말' 구분.
   - 일괄 다운로드 경로 없음(웹 검색만). 개별 확인용.
3. **최신 ICT 시사용어 2025** (TTA·전자신문 공동, PDF 무료 배포)
   - terms.tta.or.kr 공지 77번에서 다운로드. AI·양자·보안 최신 시사용어 수록.
   - 실측 결과 이미지 스캔본(225MB, 192쪽)이라 pdftotext 불가. 비전 OCR(48쪽×4 병렬)로 영한 대응 용어 73개를 추출해 glossary.tsv에 반영함(note '(ICT 시사용어 2025)').
4. **NIA 'THE AI REPORT 2024-1'** (장별 '관련 주요 용어' 절, 영한 병기)
   - 무단전재 금지·출처 표기 조건. 조건부 참고.

## 부적합·미확인

- KISA 차세대 암호(PQC) 페이지·암호이용활성화 자료실: 용어집 형식 아님.
- KQIC 양자정보 기술용어집 PDF(quantum_tech_glossary.pdf): 2026-07-18 기준 404.
- 2022 양자정보기술 백서(705쪽): 용어집 부록 없음. 2024년판 안내 URL 404.
- NIPA·ETRI 자체 용어집: 확인된 것 없음.
- TTA 표준 TTAK.KO-01.0230(양자 키 분배 용어 정의): 무료 열람 여부 미확인.

## 반영 방침

- 용어집 대량 병합은 라이선스 확인 전 보류. 개별 용어의 '표준 표기 사실'(예:
  Post-Quantum Cryptography=양자내성암호)은 확립된 표기로서 note에 참조처를 밝히고
  선별 수록한다.
- terminology.md의 번역어 결정 우선순위 참조처에 위 출처를 추가했다.
